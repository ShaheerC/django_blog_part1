DJANGO BLOG PART I

Date Created: July 25, 2019

    What is a client?

  • A device that is sending data requests to the web

    • What is a server?

  • The server stores data to send back to the client

    • What is an HTTP request?

  • A request from client to server utilizing the Hypertext Transfer Protocol

    • What is an HTTP response?

  • The response from the server to the clients request utilizing the Hypertext Transfer Protocol

    • What is Django and why do we use it?

  • Django is an open-source python web framework used for rapid development
